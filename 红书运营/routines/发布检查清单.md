# 发布检查清单（发布 April）

在发帖前逐项确认（人工最终点一次「发布」）。

## 文案

- [ ] 无绝对化医疗承诺、无未授权金融引导  
- [ ] **心理向**：无对读者的精神障碍「诊断」标签、无「包治/一定好转」等疗效承诺；自称咨询相关时资质与平台规则一致  
- [ ] 引流话术符合小红书规范（微信/外链等按平台规则）  
- [ ] 弱求职内容未超过 `aboutme.md` 约定频率  
- [ ] 无优绩主义、无假疗愈套话（对照 `aboutme.md` 不做清单）  
- [ ] **观看/阅读/收听渠道** 仅为 **正版或官方平台**；无盗版站、无「全集免费观看」类表述  
- [ ] 含 **地区/会员/上架可能变化** 的免责提示（与 `aboutme` 或模板一致）

## 标题与标签

- [ ] 标题与正文一致，无标题党到「文不对题」  
- [ ] 话题标签数量合理（按你账号习惯，常见约 3～8 个）

## 图片

- [ ] 清晰度足够（非糊图）  
- [ ] 若含第三方剧照/专辑封面：已评估风险或已用字卡替代  
- [ ] 封面主文案可读（对比度、安全区）

## 发布后记录

- [ ] 在 `发布日志.md` 填写：发布时间、笔记链接、当日日记路径  
- [ ] 在 `内容库/published/` 新增一条归档元数据文件
