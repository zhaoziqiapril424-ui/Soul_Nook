# 网络采风 URL 白名单（维护者编辑）

> 选题 April 使用 MCP Fetch 时：**仅允许**访问本文件列出的 **域名**（或前缀）。每行一个；以 `#` 开头为注释。  
> 可按需增删；**不要**写入需登录才可见的私有链接。

## 示例域名（请按你实际采风来源修改）

```
# 百科 / 元数据
zh.wikipedia.org
en.wikipedia.org
www.wikipedia.org

# 书
book.douban.com
read.douban.com

# 影
movie.douban.com
www.imdb.com

# 音 / 官方艺人或专辑公开页（示例，按你常用改）
music.douban.com
www.youtube.com
y.qq.com
music.163.com

# 流媒体「作品是否在架」查询类公开页（示例；勿用需登录的个性化推荐接口）
www.justwatch.com
```

## 临时扩容

若某日需在对话中允许额外域名，由你在主会话里 **明确写出本次允许的域名**，并由你把常用域 **合并进本文件** 后再自动化。
