## 一、账号定位（运营真源）

### 1.0 商业与转化（心理咨询 IP）

- **IP 方向**：账号对外定位为 **心理咨询向 IP**——用小红书内容 **建立信任与专业感**，再 **引导意向用户了解或预约心理咨询服务**（转化是主目标之一，与「可读的内容」并行）。  
- **转化路径（请把下面改成你的真实链路）**：示例：笔记文末 **软性 CTA** → **私信关键词 / 简介链接 / 店铺或表单** → **首次知情沟通或预约**。具体话术、价格与排期 **不要写进仓库**（避免泄露隐私与商业细节），只在个人执行清单里维护。  
- **内容与咨询的自然接缝**：日更仍以 **书或影 + 情绪母题**（见 1.2）为壳，但选题与观点要 **能接到「关系、压力、自我价值、边界、丧失与适应」等可被咨询情境承接的主题**——避免只做影评读后感而与「为什么来找我做咨询」毫无关系。  
- **合规与表达边界（公域硬约束）**  
  - **科普与陪伴 ≠ 个体咨询**：笔记是 **普及心理学常识、情绪命名、资源与视角**；**不对读者做精神障碍「诊断」**，不说「你这就是××症」。  
  - **资质自述**：若你本人或团队 **持证/受训** 可执业，在简介与必要处 **按监管部门与平台规则** 写明；**若无相应资质**，内容仅限个人成长分享与一般性信息，**禁止冒充心理咨询师或医疗机构**。  
  - **转化话术**：可写「了解预约方式／是否适合咨询」，**禁止** 「包治」「一定好转」「替代治疗」等 **疗效承诺**；危机情况引导 **当地热线或线下急诊**（具体号码以你对外统一口径为准，不必写入仓库）。  

### 1.1 主叙事：心灵角落（内容外壳）

- **帮谁**：觉得累、卷、对未来不安，仍想保留 **一小块能喘气、能被听懂** 的读者（也是过去的自己）；其中一部分 **愿意进一步通过心理咨询深入梳理**。  
- **提供什么**：每天一个 **很小的入口**——**一本书或一部电影**（日更主形态二选一，见 1.2）+ **1 段碎碎念** + **1 个提问** + 同条里的 **可核验 BGM/OST** 与 **正版书影音渠道**；不灌假疗愈、不造「你必须每天自愈」的 KPI 压力；**文末可按模板增加「了解咨询」类软性引导**（见 `角落栏目模板`）。  
- **我是谁（写作人格）**：ENFJ、爱观察、爱碎碎念；语气像 **朋友递水**——承认局限、邀请评论补一条真实。别人记住你：**通人性、周到、不装全知**；正文须有 **第一人称** 的站队或回忆接口；**心理向 IP 时**：专业处 **简洁、可核对**，不堆术语吓人。  

### 1.2 固定栏目（日更默认）— 书或影 + 配乐/相关乐

- **栏目名**：角落一分钟（可改名）  
- **日更主形态（你已拍板）**：**每天 1 条，主体为「一本书」或「一部电影」**（二选一，由当日选题定）；**不要**「纯歌单日」与书影日混成同一套栏目名，避免标签漂移。  
- **选题回避（你最新要求，选题 April 必遵守）**  
  - **主动跳过**「知名度/国民度极高、人人皆知的」书与影（如常年霸榜的必提经典、全球现象级 IP、被自媒体写滥的例句名场面）。具体可根据：豆瓣/维基「标记人数」、是否**近三年出版/公映的独立或作者向作品**、**非好莱坞年度票房前五** 等作辅证；**宁可选稍冷门但可讨论空间大的**，不选「只写金句无新知」的教材级名著。若候选撞车常见清单，**换一批**。  
- **内容深度（真知灼见，不是列清单）**  
  - 文案**禁止**只写：剧情梗概、百科式简介、无观点的作品清单。必须至少有：**一句明确观点** 或 **一个未盖棺的矛盾** 或 **一条可反直觉的读法/看法**，并写清「**这和今天的情绪、和读者处境有什么关系**」。可引用 1 处原文/台词，但要为 **「观点」** 服务。  
- **个人色彩**  
  - 正文须出现 **第一人称**（我/我们）的 **具体站队或回忆接口**（可基于真实、也可「写作人格」下的一致口气），**避免** 第三方解说腔；可固定 2～3 个你的口头禅/句式（在下方自填一句）。  
- **配图 / 书影插图（你已更新偏好）**  
  - **主封面（必需）**：**治愈风**；画面里要有 **与当日书/影相关的角色或意象**（用 **原创再诠释**：水彩/插画/柔光剪影等，**禁止** 直接贴官方剧照、禁 1:1 复刻海报人物构图以免版权争议）。  
  - **备选内页（可选）**：**与主封面同一治愈系风格** 的 **背景 + 文字排版**（金句/小标题），保证一组图视觉统一。  
  - **生图工具**：默认 **即梦（网页/App）人工出图**——**不需要** 为即梦单独接 API 或 MCP；把 `设计稿.md` 里的中英文提示词复制进即梦即可，导出后按日期放入 `assets/generated/YYYY-MM-DD/`。  
  - **若将来要「Agent 自动改图」**：仅当你有 **即梦或其它平台的官方 HTTP API** 与 Key 时，才用 **Bash + 环境变量** 或 **对应 MCP** 自动化；未配置前 **全部手动作业即可**，不必为「能发笔记」额外接 API。  

- **音乐部分（同一条笔记里呈现）**  
  - **优先**：**电影**则优先 **原声带 OST / 官方发行配乐** 中的曲目；**书**则选 **与书中情绪或改编影视强相关、且可核验口碑** 的曲（如原著改编电影 OST、作者时代/同主题名曲）。  
  - **「好评率/口碑高」的约定**：选题/文案 April 在写推荐时，**至少给出一条可引用依据**（例如：网易云/QQ 音乐 **收藏量或热评高**、Spotify/Apple Music 可见数据、或权威乐评/奖项提名），并注明 **数据为发布日检索快照，会变化**。不编造「全网第一」类话术。  
  - **条数**：**1 首主推荐 BGM** + **0～2 首备选**（可标注「前奏更适合当口播 BGM」等）。  
- **与画面形式**：若发 **视频笔记**，**BGM 优先用小红书内置音乐库** 里能搜到的同名或翻奏（版权合规）；若曲库无该曲，在文案中说明「建议搜索关键词 ___」并降低对「必须原曲」的承诺。  
- **基础结构仍保留**：1 段碎碎念 + 1 个开放式提问 + 可选一句金句；**「在哪里读/看/听」** 小节须同时覆盖 **书/影的正版渠道** 与 **音乐的正版收听渠道**。  
- **无人拍板时默认方案**：若当日未在对话中明确「用方案 2 / 方案 3」等，则 **一律采用《选题建议》中的「方案 1」（第一套）** 进入后续 **文案 April → 设计 April**（与口令「今天发什么」一键工作流对齐）。

### 1.3 周更

- **频率**：建议 **每周 1～2 篇** 加长（多图大纲 + 分段小标题），冲收藏与深度关注。  
- **说明**：由选题在方案中标注 **「是否建议今日即周更」**；不强行与日更同频。  

### 1.4 副线规则

旅行、美食、金句、生活碎片 **仅当**能回答：「它和今天这个『角落』里的情绪，有什么关系？」——否则不发或极低频。

### 1.5 求职 / 行业（弱关联）

- **上限**：**每周不超过 1 次** 出现求职/行业联想，或 **仅** 在结尾提问里点到，**不** 单独成篇做第二主叙事。  
- **可做边界**：**仅** 在与当日书/影/情绪自然接壤处出现；**不做** 求职陪跑、改简历、大厂优绩路径贩卖。  

### 1.6 不做清单（硬）

1. 短期不做纯追星；不做与主叙事无关的带货测评。  
2. 不做优绩主义叙事。  
3. 不做假疗愈。  
4. 不做无法核验的行业/求职断言（须：现象 + 来源 + 欢迎纠偏）。  
5. **心理向补充**：不在公域 **诊断** 他人或贴精神障碍标签；不做 **医疗/疗效承诺** 式引流；无资质则不冒充专业咨询身份。

## 一点五、观看与收听习惯（给选题 / 文案 April 写「正版渠道」用）

- **主要受众地区**：以 **中国大陆** 读者为主；若你有海外粉，文尾可一句「其他地区请用本地流媒检索同名」。  
- **默认假设**：正文以 **国区 App 在架与会员前提** 为主写「在哪里读/看/听」；标注 **片单/上架会随版权变化**。  
- **书**：微信读书、豆瓣阅读、多抓鱼/纸书、当地图书馆/微信图书馆等——**以读者手机能搜到为准**；选题/文案不承诺永久在架。  
- **影剧**：爱奇艺、腾讯视频、优酷、B 站（版权片）、咪咕等；有 **租赁/单片** 的写清。海外片写 Netflix / Disney+ 等时注明 **地区与片库可能不同**。  
- **音乐（含 OST / 高口碑单曲）**：网易云、QQ 音乐、Apple Music、Spotify；**电影优先 OST 官方曲名+专辑**；**口碑**须附可核验依据（见 1.2「好评率」约定），不造「全网第一」话术。  
- **免责声明模板**（可粘贴进文）：「片单/书单/上架情况会随版权与地区变化，请以你手机 App 内检索为准；若我标错欢迎指正。」  
- **发视频时**：优先说明「剪辑时请在小红书音乐库中搜索：___（或相近风格）」，降低版权风险。
