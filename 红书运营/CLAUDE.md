# Claude Code — 小红书「心灵角落」运营仓

## 设定加载顺序（给 Claude / 子 Agent）

每轮任务建议按此顺序读上下文，避免漏规则：

1. **`aboutme.md`** — 账号真源、口癖、日更/周更/求职边界、**无人拍板 → 方案 1**、配图与即梦约定。  
2. **`CLAUDE.md`**（本文件）— 触发语、禁止项、发帖边界。  
3. 当日 **`角落素材/YYYY-MM-DD-raw.md`**（若有）。  
4. 任务相关 **`routines/*.md`**（一键口令、采风规则与白名单、栏目模板、检查清单等）。  
5. **Subagent** 专精提示见 **`routines/agents.json`**（以 `claude --agents …` 注入）。

## 仓库根

本目录即 **`红书运营/`** 项目根。所有路径相对本文件所在目录。

## 本地 / Web 工作区怎么用（不搞「假一键 Run」）

- **Web IDE / 浏览器里的 Claude Code**：把**工作区根**指到含有本文件的目录（与 `README.md` 同级）；在**对话里**使用下方「触发语」表格即可驱动内容写入 `日记/` 等。**是否**有「对某个 `.md` 点 Run」取决于具体产品，本仓不承诺统一入口——**`routines/每日操作清单.md` 是给人对照的步骤清单**，不是各环境通用的可执行脚本。
- **本地 `claude` CLI**：在**同一目录**打开终端，按「Subagents」小节的命令启动；若走国内网关，需先按 **[`docs/协作者从clone到第一句话.md`](docs/协作者从clone到第一句话.md)** **B** 路径配置环境变量。切勿使用 `export $(cat ... | xargs)` 这类方式加载密钥文件（易截断、不安全），应 `source` 由模板生成的 env 文件，或依赖官方登录凭据。
- **`routines/bootstrap_after_clone.sh`**：只负责创建常用文件夹；**不读取** `~/.secrets`（若未找到该文件会打印一句提示，属正常）。

## 主叙事（以 aboutme 为准）

- **唯一主叙事**：心灵角落作为 **内容外壳**（书/影/歌 + 碎碎念 + 提问；拒绝假疗愈与优绩叙事）；账号整体定位为 **心理咨询向 IP**，**内容引流 → 了解/预约心理咨询** 见 `aboutme.md` §1.0，合规边界见 §1.0 / §1.6。
- **必读**：`aboutme.md`（每次任务前读）。
- **可选舵向**：`角落素材/YYYY-MM-DD-raw.md`（心情、指定作品、禁忌；可为空）。  
- **网络采风**：选题 April 使用 **MCP Fetch**（如 Smithery `https://fetch--smithery-ai.run.tools`）仅在 **`routines/网络采风白名单.md`** 内域名拉取公开页，整理书单/影单/歌单并写清 **正版观看/阅读/收听渠道**；细则见 **`routines/网络采风规则.md`**。**禁止** 盗版链接、登录爬取、破解付费墙。

## 触发语 → April

| 你说 | 动作 |
|------|------|
| 「早上好」「开始今天的工作」 | 确保存在 `日记/YYYY-MM-DD/`，运行 **选题 April**（或 Subagent `topic-april`）→ `选题建议.md` |
| 「用方案 N」「选题定第 N 个」 | **文案 April** → `文案初稿.md` |
| 「文案定稿，做图」 | **设计 April** → `设计稿.md` + `assets/generated/` 说明 |
| 「终稿，走发布检查」 | **发布 April** → 检查表 + `发布日志.md` + 内容库索引（**不自动发帖**） |
| 「数据已贴，写复盘」 | **复盘 April** → `数据复盘.md`；**仅追加** `aboutme.md` 的「## 账号迭代记录」 |

## 禁止项

1. 未经白名单与规则允许的 **HTTP 请求** 不做；不把 Cookie/Token 写入仓库或 Markdown；不对需登录或付费墙后的页面做自动化抓取。  
2. 默认 **不调用发帖 API**、不模拟登录小红书后台。  
3. 文案遵守 `aboutme.md` 中的 **不做清单**；弱求职内容遵守 aboutme 中的 **频率/篇幅上限**。  
4. 不覆盖 `aboutme.md` 全文；复盘只追加约定小节。

## Subagents

配置见 `routines/agents.json`。启动示例（在本目录下）：

```bash
claude --bare --agents "$(cat routines/agents.json)"
```

## 配图与自动生成插图（是否必须 MCP / API？）

- **手动画图（默认）**：用 **即梦** 等工具打开 `设计稿.md` 中的提示词出图，**不需要** 接任何 API 或 MCP。  
- **自动出图（可选）**：仅当配置了某平台的 **生图 API Key** 时，才用 **Bash** 或 **该平台的 MCP** 让 Agent 写文件到 `assets/generated/`。  
- **即梦**：若只做账号内人工生成，**无 API 也能完成整条内容链路**；只有你要「无人工点即梦」时才去查即梦/火山是否开放 **开发者 API** 并配 Key。  
- 版权：不 1:1 复刻官方海报/剧照；见 `aboutme` §1.2 与 `.secrets/README.md`。

## 一键口令（日更总控）

- 说 **「今天发什么」**：按 `routines/一键口令工作流.md` 尽量串完「选题 → 文案 → 设计」；`aboutme.md` 里需写清 **是否自动选定方案**（否则只生成到 `选题建议.md` 并停）。  

## 定时自动「只选题」

本机非交互：见 **`routines/每日选题自动化.md`**（`daily_run_topic_april.sh` + launchd 示例）。默认每天固定时刻只写 `日记/…/选题建议.md`；不替代人工审发。
