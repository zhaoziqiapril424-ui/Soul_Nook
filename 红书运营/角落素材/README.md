# 角落素材 — 每日投喂（合规来源）

## 用法

**可选**：新建或编辑 **`YYYY-MM-DD-raw.md`**（日期 = 你要发笔记的当天或策划当天），用于 **心情 / 指定作品 / 禁忌** 舵向。  
若当日文件为空或缺失，**选题 April 仍会**在 `网络采风白名单.md` 约束下通过 **MCP Fetch** 自主搜集书影音候选，并在 `选题建议.md` 中写明 **正版观看渠道**。

## 建议结构（复制到当日文件）

```markdown
# 角落素材 · YYYY-MM-DD

## 今日心情（一句话）

## 想用的作品（三选一或组合）

- 书：
- 影：
- 歌：

## 链接 / 摘录来源（可选）

- （只贴你有权使用或合理引用的来源）

## 给 April 的备注（可选）

- 例如：想弱关联求职 / 想偏治愈 / 想留白封面
```

## 注意

- **不**在此文件放账号密码、Cookie、API Key。  
- 歌词、台词 **长引用** 注意版权与平台习惯；优先短摘 + 自己的话。
