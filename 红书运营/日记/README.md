# 日记 / `YYYY-MM-DD/`

## 命名

- 一级目录名：**ISO 日期** `YYYY-MM-DD`（与小红书笔记发布日一致，便于检索）。

## 单日文件（默认）

| 文件 | 说明 |
|------|------|
| `选题建议.md` | 选题 April |
| `文案初稿.md` | 文案 April |
| `设计稿.md` | 设计 April |
| `发布日志.md` | 发布 April |
| `数据复盘.md` | 复盘 April（可次日再建） |

## 状态（可选在当日目录加 `状态.md`）

示例：`选题已定` → `文案已定` → `设计已定` → `已发布` → `已复盘`。
